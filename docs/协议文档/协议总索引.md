## 通信规格

### USB规格

- **有线**: VID=0x1A86, PID=0x8312
- **无线**: VID=0x1A86, PID=0x8300
- **机型名称**: Game Mouse
- **协议通道**: HID → Interface 2 → EP3 → Usage Page (0xFF13)
- **数据包长度**: 64字节（不足补0）

### 重传机制

- PC端检测ACK异常时最多重传3次
- 重传间隔100ms

## 开发优先级建议

### 第一阶段（核心功能）

1. **M05 设备信息获取** - 初始化必需，获取设备能力参数
2. **M02 基础设置读取** - 同步设备当前配置
3. **M01 基础设置配置** - 核心配置功能

### 第二阶段（扩展功能）

4. **M06 电池信息读取** - 无线设备必需
5. **M03 宏配置** - 高级功能

### 第三阶段（系统功能）

6. **M07 恢复出厂设置** - 系统维护功能

### 暂缓开发

- **M04 宏读取** - 协议暂不开发
- **M08 固件升级** - 协议暂不开发
